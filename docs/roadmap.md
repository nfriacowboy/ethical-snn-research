# Project Roadmap

## Overview
Multi-phase investigation of ethical behavior emergence in SNNs using dual-process theory.

---

## Phase 1: Dual-Process Architecture (Current)
**Timeline**: January 2026 - April 2026

### Goals
- Implement survival-only SNN (SNN-S)
- Implement ethical SNN (SNN-E)
- Compare behavioral outcomes in grid-world simulations

### Milestones
- [x] Pre-registration on OSF
- [ ] Environment implementation (Week 1-2)

- [ ] Manuscript preparation (Week 11-14)

### Deliverables

---

## Phase 2: Multi-Level Selection (Planned)
**Timeline**: May 2026 - September 2026

### Goals
- Implement group-level selection pressures
- Test altruistic behavior emergence
- Explore kin selection vs. group selection

### Hypotheses
- H2.1: Group selection promotes cooperation
- H2.2: Ethical networks evolve stronger in group contexts
- H2.3: Kin recognition emerges from network structure

### Key Changes
- Multi-organism families/groups
- Genetic inheritance mechanisms
- Inter-group competition scenarios

---

## Phase 3: Petri Net Integration (Planned)
**Timeline**: October 2026 - February 2027

### Goals
- Model ethical reasoning as Petri net processes
- Integrate symbolic reasoning with neural processing
- Test hybrid neuro-symbolic decision-making

### Technical Approach
- Petri nets represent ethical rules/norms
- SNN outputs activate Petri net transitions
- Petri net state modulates SNN processing

### Research Questions
- Can symbolic ethics guide emergent behavior?
- How do learned rules interact with innate drives?
- Does hybrid architecture improve generalization?
